# OnlineClasses
Make your online classes much easier 

## Instalation
### Prerequisites  
* FFmpeg
* Python 3
* pip

### Linux
```python
pip3 install -r requirements.txt
```
### Windows
```python
pip install -r requirements.txt
```

## Usage
### Linux
```python
python3 main.py
```
### Windows
```python
python main.py
```
